Расшифровка номера сборки:

Пример: (ABCD.EE-999)

  A - тип сборки (0 - тест, 1 - бета, 2 - пререлиз, 3 - релиз)
  В - версия файловой системы
  C - версия синтаксиса
  D - версия UI
  EE - версия наполнения (двухзначное число)
  999 - порядковый номер сборки (трехзначное число)
