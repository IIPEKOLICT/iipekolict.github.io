Расшифровка номера сборки:

Пример: (A-BB.CC.DD.999)

  A - тип сборки (А - тест, B - бета, R - релиз)
  BB - версия синтаксиса
  CC - версия UI
  DD - версия наполнения (двухзначное число)
  999 - порядковый номер сборки (трехзначное число)
